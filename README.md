# DU CAID Brand Site (GitHub Pages)

This is a ready-to-deploy, responsive static website for `ducaid.github.io` using DU colors:

- DU Crimson: `#BA0C2F`
- DU Gold: `#A89968`
- White: `#FFFFFF`
- Black: `#000000`

## Deploy
1) Create org repo `ducaid.github.io`.
2) Push these files to `main`.
3) In **Settings â†’ Pages**, ensure "Build and deployment" = **GitHub Actions**. The provided workflow deploys automatically.

## Customize
- Edit text in `index.html`, `about.html`, `projects.html`, `people.html`, `contact.html`.
- Adjust colors in `assets/css/styles.css` under `:root` if needed.
- Replace `assets/img/favicon.svg` with your logo (same filename to keep link).

## Accessibility
- High-contrast hero uses Crimson with light text.
- Buttons meet color-contrast on both light and dark backgrounds.
